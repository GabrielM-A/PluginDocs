Welcome ​

This plugin is designed to streamline and enhance the interaction system in Unreal Engine, offering an intuitive and customizable approach to interactions within your project. Below, you’ll find detailed documentation to help you get the most out of the plugin. If you have any questions or feedback, feel free to reach out – I’m always happy to assist!

Simple Setup ​

Follow these simple steps to get the Interaction Plugin up and running in your project.

  1. Attach the Interaction Component to any player character.
  • This component handles the player's ability to interact with objects in the game world.

Interaction Component in Third Person Character BP

  1. Attach the Interactable Component to any Actor you want to be interactable.
  • This component defines the object the player can interact with, such as a door, button, or pickup item.

Interactable Component in BP_Object

  1. Attach the Interactable Interface

And that’s it! Your basic interaction setup is ready to go. ​
